# SQL Sales Analysis

## Objective
Analizar datos de ventas por país y categoría para entender el rendimiento global del negocio.
This project includes SQL queries to analyze sales performance and customer behavior.

## Dataset

The dataset consists of three tables:

- orders: contains information about each order (order_id, customer_id, product_id, amount)
- customers: contains customer information (customer_id, country)
- products: contains product categories (product_id, category)

## Key Queries

- Total sales by country
- Total sales by category
- Top 3 customers by total spending
- Customers with at least 2 orders
- Highest order per customer
- Top 2 orders per customer
- Countries with total sales greater than 1500

## Tools Used

- SQL (SQLite)
- DB Browser for SQLite
- Visual Studio Code
